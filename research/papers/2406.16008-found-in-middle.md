# Found in the Middle: Calibrating Positional Attention Bias Improves Long Context Utilization

> Source: arXiv:2406.16008v2 (3 Jul 2024, full HTML fetched 2026-06-29). Hsieh, Chuang, Li, Wang, Le, Kumar, Glass, Ratner, Lee, Krishna, Pfister (U Washington / MIT / Google Cloud AI). ACL Findings 2024. PDF in pdf/.

## Thesis

Lost-in-the-middle is not just a data quirk: it is driven by an intrinsic **U-shaped positional attention bias**. Tokens at the start and end of the prompt get more attention regardless of relevance, so mid-context facts are under-used. Calibrate the bias away and the same model can find the middle.

## Method

Reproduce lost-in-the-middle on Vicuna-7b-v1.5-16k, K=20 docs, gold at position 10. Model `Attn(x,k)=rel(x)+bias(k)+ε`; estimate bias via a dummy document, subtract to get **calibrated attention**. Rescale attention on last 16 of 32 layers; pure inference, greedy, 2× A100. Datasets: NaturalQuestions (2655 q), SynthWiki (990 fictional Wikipedia, contamination-free), K={10,20}.
