# 2603.07670 — Memory for Autonomous LLM Agents: Mechanisms, Evaluation, and Emerging Frontiers

> Source: <https://arxiv.org/abs/2603.07670> — Pengfei Du et al. (8 Mar 2026). PDF in `pdf/2603.07670.pdf`.

## Thesis

A survey of how memory turns a stateless LLM into an adaptive agent. Models hold knowledge only in frozen weights and start each request from the prompt alone, so long-running agents need runtime/persistent stores external to the weights. The survey organises memory by **time horizon** — working (live scratchpad), episodic (recent events), semantic (durable facts), and procedural (learned skills/policies) — and surveys mechanisms for the **write → manage → read** cycle, evaluation, and open frontiers (reflective and policy-driven memory). A recurring finding: the performance gap between an agent with good memory and one without can exceed the gap between model versions.
