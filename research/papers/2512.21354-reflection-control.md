# Reflection-Driven Control for Trustworthy Code Agents

> Source: arXiv:2512.21354v1 (22 Dec 2025, full HTML). Wang, Quan, Yu, Hu, Tsang. Reflex layer in Plan–Reflect–Verify.

## Thesis

Reflection from post-hoc patch → first-class internal control loop. Reflex Module (plug-and-play): self-check router (SAFE/UNSAFE), reflective prompt engine (multi-turn CoT), reflective memory (ChromaDB dynamic + static secure-coding RAG). Gate before commit + machine-verifiable evidence (auditable).

## Results (8 CWE, CodeQL, 4 LLMs, 25×5)

- Sec.Rate gains: +2.9 to +11.2pts (e.g. 83.7→94.9). Pass largely preserved. Dynamic RAG sim 0.85→0.98, success 85→100%, fallback 15→0% by R4. >0.70 sim → 93.8% fix; 0.95–1.0 → 100%. Single reflection round = ~90% of gains. Overhead: 28.8s/scene, LLM 84.4%, RAG 0.8s; $5.37e-4/scene.

## Use

Ch3/Ch5: in-loop reflection > post-hoc; evidence-grounded secure-codegen; diminishing returns after 1 round.
