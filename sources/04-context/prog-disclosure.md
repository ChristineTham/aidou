A classic example of this pattern is the print dialog in [macOS](/wiki/MacOS "MacOS"). While printing a page, the print dialog shows only a small subset of choices. If the user wants more advanced options, they can click the "Show Details" button to reveal these features in a secondary screen.<sup>\[*[<span title="This claim needs references to reliable sources.
