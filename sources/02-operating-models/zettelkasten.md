## Literary references
- In chapter two of [Robert M. Pirsig](/wiki/Robert_M._Pirsig "Robert M. Pirsig")'s philosophical novel *[Lila: An Inquiry into Morals](/wiki/Lila:_An_Inquiry_into_Morals "Lila: An Inquiry into Morals")* (1991), the main character describes an index card system of notes he's keeping for a book. While the German word <span title="German-language text">*Zettelkasten*</span> isn't used, the descriptor "slips" is used repeatedly (as opposed to index card which appears four times) and the system has the general form and function of a card file as commonly used by writers.
