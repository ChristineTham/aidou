## Common software measurements
Common software measurements include:
- [ABC Software Metric](/wiki/ABC_Software_Metric "ABC Software Metric")
- [Balanced scorecard](/wiki/Balanced_scorecard "Balanced scorecard")
- [Bugs](/wiki/Software_bug "Software bug") per line of code
- [Code coverage](/wiki/Code_coverage "Code coverage")
- [Cohesion](/wiki/Cohesion_(computer_science) "Cohesion (computer science)")
- [Constructive Cost Model](/wiki/COCOMO "COCOMO")
- [Cyclomatic complexity](/wiki/Cyclomatic_complexity "Cyclomatic complexity") (McCabe's complexity)
- Defect potential - expected number of defects in a particular component
- Defect removal rate
- [DSQI](/wiki/DSQI "DSQI") (design structure quality index)
- [Halstead Complexity](/wiki/Halstead_complexity_measures "Halstead complexity measures")
- [Instruction path length](/wiki/Instruction_path_length "Instruction path length")
- [Maintainability index](/wiki/Maintainability#Software_engineering "Maintainability")
- [Source lines of code](/wiki/Source_lines_of_code "Source lines of code") - number of lines of code
- [Program load time](/wiki/Loader_(computing) "Loader (computing)")
- [Program size (binary)](/wiki/Binary_file "Binary file")
- WEAVE score
- [Weighted Micro Function Points](/wiki/Weighted_Micro_Function_Points "Weighted Micro Function Points")
## Limitations
- [Scheduling](/wiki/Schedule_(project_management) "Schedule (project management)")
- [Software sizing](/wiki/Software_sizing "Software sizing")
- [Programming complexity](/wiki/Programming_complexity "Programming complexity")
- [Software development effort estimation](/wiki/Software_development_effort_estimation "Software development effort estimation")
- [Software quality](/wiki/Software_quality "Software quality")
A specific measurement may target one or more of the above aspects, or the balance between them, for example as an indicator of team motivation or project performance.
